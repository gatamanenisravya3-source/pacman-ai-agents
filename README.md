# Pacman Game — AI Agents

Aug 2024 – Dec 2024 · George Mason University

AI agents for the Pacman environment across three paradigms:
- Reinforcement Learning — Q-Learning, Value Iteration
- Multi-Agent Search — Minimax, Alpha-Beta Pruning, Expectimax
- Probabilistic Inference — Bayesian networks, particle filtering, HMMs for tracking hidden entities under noisy data

Also implemented A*, BFS, DFS, and Reflex Agents for maze-solving, with custom evaluation functions for adversarial scenarios involving ghosts. Completed Ghostbusters, Multi-Agent Search, and Reinforcement Learning projects, validated with autograder-based testing.

## Tech Stack
Python, Reinforcement Learning, Bayesian Networks, Search Algorithms
